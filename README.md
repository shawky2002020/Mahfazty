## 📜 **Overview**

**Mahfazty** is a sleek, user-friendly web application crafted to simplify expense tracking. With a focus on intuitive design and functionality, **Mahfazty** empowers users to:

- Track their income and expenses.
- Categorize spending for better organization.
- Gain valuable insights into their financial habits through interactive analytics.

Whether you're looking to manage personal finances or monitor household expenses, **Mahfazty** delivers a seamless and efficient experience across devices.

---

## ✨ **Key Features**

- **📊 Expense Management**: Add, edit, and delete expenses effortlessly.
- **📅 Monthly Reports**: Access detailed summaries of your spending habits on a monthly basis.
- **🗂 Categorization**: Organize your expenses into predefined or custom categories for clarity.
- **📈 Interactive Charts**: Gain insights into your spending patterns with vibrant pie charts and bar graphs.
- **🌐 Responsive Design**: Optimized for desktops, tablets, and mobile devices.
- **🔐 Secure Storage**: Ensures data privacy and reliability.

## 🛠️ **Tech Stack**

### **Frontend**:

- Angular
- HTML5, CSS3, TypeScript

### **Backend**:

- Node.js with Express.js
- RESTful APIs

### **Database**:

- MongoDB (NoSQL Database)

### **Additional Tools**:

- Git for version control
- npm for package management
- ngx-charts for interactive data visualization

## 🚀 **Getting Started**

Follow these instructions to set up and run **Mahfazty** on your local machine.

### **Prerequisites**

Ensure the following tools are installed on your system:

- [Node.js](https://nodejs.org/) (LTS version)
- npm (comes bundled with Node.js)
- MongoDB (installed and running locally)
